# 📌 오날입 (오늘 날씨에 입을 것)

오늘의 날씨를 기반으로 **상의, 하의, 신발을 추천해주는 웹사이트**입니다.  
사용자의 현재 위치를 자동으로 감지하여, 실시간 날씨에 맞는 코디를 제안합니다.

## 🌤️ 프로젝트 소개

"오날입"은 날씨에 따라 매일 옷을 고민하는 사람들을 위해 만들어진 서비스입니다.  

현재 위치의 날씨 데이터를 활용하여  
👉 오늘 입기 좋은 옷을 자동으로 추천합니다.

---

## 🚀 주요 기능

### 📍 위치 기반 날씨 정보 제공
- Geolocation API를 통해 사용자 위치 자동 감지
- 현재 지역의 실시간 날씨 정보 표시

### 👕 오늘의 코디 추천
- 날씨 및 온도에 따른 의류 추천
  - 상의
  - 하의
  - 신발

### 🌡️ 날씨 기반 추천 로직
- 온도 및 기후 조건에 따른 추천
  - 25도 이상 → 반팔, 반바지, 샌들
  - 15~25도 → 긴팔, 청바지, 운동화
  - 5~15도 → 니트, 자켓, 운동화
  - 5도 이하 → 패딩, 두꺼운 바지, 부츠
  - 비 → 우산 및 방수 신발 추천
  - 눈 → 부츠 추천

### 🎨 UI/UX 기능
- 다크모드 / 라이트모드 지원
- 반응형 디자인 (모바일 대응)
- 직관적인 카드형 UI
